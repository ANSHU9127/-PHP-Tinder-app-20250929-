# -PHP-Tinder-app-20250929-
MatchMate is a modern Tinder-style mobile application built using React Native (Expo) for the frontend and Laravel 11 for the backend. The app displays recommended people in a swipe-deck UI, where users can swipe right to like and left to dislike, just like the Tinder interface. The project follows Atomic Design, uses React Query for server state .
